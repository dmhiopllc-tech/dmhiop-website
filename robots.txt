## 🎨 Design System

### Color Palette

```css
Primary Teal:   #62f5e7  /* Buttons, accents, hovers */
Primary Brown:  #815956  /* Logo text, headings */
Background:     #fff     /* White background */
Text Dark:      #333     /* Main text */
Text Light:     #555     /* Secondary text */
```

### Typography

- **Primary Font**: Open Sans (body text, 300-800 weights)
- **Secondary Font**: Mukta (headings, 200-800 weights)
- **Logo Font Size**: 1.1rem
- **Heading Sizes**: h1 (2.5rem), h2 (2rem), h3 (1.5rem)

### Hero Banner

- **Background Image**: Sandia Mountains (`/sandia-banner-medium.jpg` for desktop, `/sandia-banner-mobile.jpg` for mobile)
- **Gradient Overlay**: `linear-gradient(rgba(98, 245, 231, 0.7), rgba(129, 89, 86, 0.7))`
- **Height**: 350px desktop, 250px mobile
- **Text Color**: White (#fff)

### Logo

- **Image**: `/dmh-logo (1).png`
- **Size**: 50px × 50px
- **Text**: "Desert Mountain Healing IOP" in brown (#815956)
- **Layout**: Flexbox with 0.75rem gap

## 🔍 SEO Configuration

### Sitemap (sitemap.xml)

- **Location**: `https://dmhiop.com/sitemap.xml`
- **Format**: XML
- **Pages**: 11 public pages
- **Update Frequency**: Homepage/Reviews (weekly), Others (monthly)
- **Last Modified**: April 23, 2026

### Robots.txt

- **Location**: `https://dmhiop.com/robots.txt`
- **Strategy**:
  - ✅ **Allow**: All 11 public marketing pages
  - ❌ **Block**: Client portal, admin pages, internal app pages
  - ❌ **Block**: Development files, backups, configuration files
  - ❌ **Block**: Folders: /js/, /css/, /scripts/, /SQL/, /__static/
- **Crawl Delay**: 1 second (respectful crawling)

### Structured Data (JSON-LD)

Each page includes:
- Organization schema with contact information
- Medical Organization schema
- BreadcrumbList for navigation
- WebPage schema with descriptions
- Local Business schema with geo-coordinates
- Provider Person schemas (team page)
- Review aggregate ratings (reviews page)

### Meta Tags

All pages include:
- Title (50-60 characters)
- Description (150-160 characters)
- Keywords (targeted local and service keywords)
- Open Graph tags (Facebook, LinkedIn)
- Twitter Card tags
- Geo-targeting (Rio Rancho, NM coordinates)
- Canonical URLs
- Voice search optimization tags
